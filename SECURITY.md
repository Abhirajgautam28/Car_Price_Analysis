Security Policy

If you discover a security vulnerability in this repository or the data it uses, please report it privately to `abhirajgautam28@gmail.com` with a clear description and steps to reproduce. Do not create a public issue that discloses the vulnerability.

Please include:

- A concise description of the vulnerability and its potential impact.
- Steps to reproduce and a minimal reproducible example if possible.
- Environment details and package versions where relevant.

We will acknowledge receipt within 3 business days and provide updates as we investigate and remediate.
